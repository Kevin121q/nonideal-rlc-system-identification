# Documentation

This directory contains supporting documentation for the Non-Ideal RLC System Identification project.

The project combines experimental measurements, computational modeling, parameter reconstruction, and identifiability analysis to investigate how accurately hidden parameters can be recovered from frequency-response data.

## Contents

### theory.md

Background theory for RLC circuits, resonance, damping, source loading, parameter estimation, and model mismatch.

### experimental_setup.md

Description of the physical circuit, measurement equipment, and experimental procedure.

### methodology.md

Overview of the complete analysis workflow, including data collection, cleaning, reconstruction, and identifiability analysis.

### identifiability.md

Discussion of inverse problems, parameter estimation, uncertainty, and practical identifiability.

### project_timeline.md

Chronological development of the project.

### references.md

Books, papers, and resources relevant to the project.
