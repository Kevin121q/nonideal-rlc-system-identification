# Reports

This directory contains written technical summaries of the major stages of the project.

The reports are intended to explain the reasoning, methodology, results, limitations, and interpretation behind the experimental and computational analyses.

## Current Reports

### resistance_sweep_results.md

Summarizes the experimental resistance sweep measurements.

Covers:

* frequency-response behavior
* damping effects
* source loading
* resonance behavior
* limitations of ideal RLC assumptions

### parameter_reconstruction_results.md

Summarizes the non-ideal parameter reconstruction workflow.

Covers:

* fitted hidden resistance
* effective inductance
* effective capacitance
* residuals
* RMSE
* parameter drift
* effective-parameter interpretation

### identifiability_results.md

Summarizes the practical identifiability analysis.

Covers:

* one-dimensional RMSE sweeps
* two-dimensional RMSE maps
* uncertainty bounds
* parameter correlations
* L-C tradeoff behavior
* model mismatch interpretation

## Purpose

These reports connect the raw analysis outputs to the scientific conclusions of the project.

They are not just result dumps. Each report explains what was tested, what was found, what the limitations are, and how the result affects the broader system-identification question.
