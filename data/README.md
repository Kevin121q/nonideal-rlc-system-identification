# Data

This directory contains all datasets used throughout the project, from raw experimental measurements to processed outputs generated by model fitting and identifiability analyses.

## Structure

### raw/

Original frequency-response measurements collected from the physical RLC circuit.

Contents include:

* Resistance sweep measurements
* Repeatability measurements
* Source voltage and resistor voltage recordings

These files are preserved as close as possible to the original experimental observations.

### cleaned/

Validated and standardized datasets generated from the raw measurements.

Processing steps include:

* Column standardization
* Data-type validation
* Gain verification
* Outlier inspection
* Consistent formatting across experiments

These datasets serve as the primary inputs for analysis and model fitting.

### processed/

Results generated from analysis pipelines.

Subdirectories include:

#### experimental/

Experimental summaries and repeatability analyses.

#### reconstruction/

Parameter-reconstruction results, fitted parameters, residual metrics, and reconstruction trends.

#### identifiability/

One-dimensional and two-dimensional RMSE analyses, uncertainty estimates, and representative identifiability studies.

All processed files are generated programmatically from cleaned datasets and can be reproduced using the notebooks and scripts contained in this repository.
