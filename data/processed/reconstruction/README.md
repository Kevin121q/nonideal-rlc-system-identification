# Parameter Reconstruction

This folder contains results from fitting a non-ideal RLC model to experimental frequency-response measurements.

Estimated parameters include:

- Hidden resistance (R_hidden)
- Effective inductance (L_eff)
- Effective capacitance (C_eff)

Contents:

- parameter_reconstruction_by_run.csv
- parameter_reconstruction_combined.csv
- per_curve_fit_results.csv
- reconstruction_summary.csv
- reconstruction_trend_table.csv

These datasets are used to investigate how reconstructed parameters vary across resistance conditions and to evaluate model adequacy.
