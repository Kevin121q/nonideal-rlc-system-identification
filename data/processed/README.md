# Processed Data

This directory contains datasets generated from experimental measurements through cleaning, analysis, model fitting, and identifiability workflows.

The processed data are organized into three categories:

## Experimental

Derived quantities from frequency-response measurements, including:

- Resistance sweep summaries
- Repeatability analysis
- Source-loading metrics
- Resonance characterization

## Reconstruction

Results from fitting a non-ideal RLC model to experimental data, including:

- Effective inductance estimates
- Effective capacitance estimates
- Hidden resistance estimates
- Per-curve fit quality metrics
- Reconstruction trend summaries

## Identifiability

Results used to evaluate parameter identifiability and uncertainty, including:

- One-dimensional RMSE sweeps
- Two-dimensional RMSE surfaces
- Practical uncertainty intervals
- Parameter uncertainty summaries

All files in this directory are generated programmatically from cleaned experimental data.
