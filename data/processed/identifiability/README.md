# Identifiability Analysis

This folder contains datasets used to evaluate practical parameter identifiability.

The analysis investigates how uniquely model parameters can be recovered from experimental frequency-response measurements.

## Representative Cases

Detailed analyses are provided for:

- 47 Ω (low damping)
- 150 Ω (moderate damping)
- 470 Ω (high damping)

Each case contains:

- One-dimensional RMSE sweeps
- L-C RMSE surfaces
- R_hidden-L RMSE surfaces
- R_hidden-C RMSE surfaces

## Summaries

Summary tables provide:

- Practical uncertainty intervals
- Parameter uncertainty widths
- Best-fit parameter estimates
- Minimum RMSE values

The results demonstrate that parameters are practically identifiable within individual experiments while also revealing parameter tradeoffs and model-mismatch effects.
