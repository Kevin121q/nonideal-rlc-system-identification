# Representative Identifiability Cases

To avoid excessive duplication, detailed identifiability analyses are shown for three representative resistance values:

- 47 Ω (low damping)
- 150 Ω (moderate damping)
- 470 Ω (high damping)

Each folder contains:

- One-dimensional RMSE sweeps
- L–C RMSE surface
- Rhidden–L RMSE surface
- Rhidden–C RMSE surface

These cases illustrate how parameter identifiability changes across operating conditions.
