# Experimental Results

This folder contains summary datasets derived directly from frequency-response measurements.

Contents:

- resistance_sweep_summary1.csv
- resistance_sweep_summary2.csv
- repeatability_summary.csv
- repeatability_pointwise_comparison.csv

These files quantify:

- Resonance frequency
- Peak gain
- Bandwidth
- Approximate Q factor
- Source loading
- Measurement repeatability

The datasets provide the experimental foundation for subsequent parameter reconstruction and identifiability analyses.
