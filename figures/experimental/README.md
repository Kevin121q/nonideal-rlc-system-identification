# Experimental Figures

This folder contains figures generated directly from experimental frequency-response measurements of the non-ideal series RLC circuit.

## Resistance Sweeps

### resistance_sweep_gain_curves.png

Measured gain as a function of frequency for all tested resistances. This figure provides the primary experimental view of resonance behavior, damping effects, and changes in peak response.

### peak_gain_vs_resistance.png

Peak measured gain extracted from each resistance sweep.

### bandwidth_vs_resistance.png

Estimated resonance bandwidth as a function of measured resistance.

### q_factor_vs_resistance.png

Approximate quality factor calculated from measured resonance characteristics.

---

## Source Loading

### source_loading_resistance_sweep1.png

Measured source-voltage behavior across frequency, illustrating generator loading and non-ideal source effects.

---

## Repeatability

### gain_repeatability_R_*.png

Comparison of repeated measurements for each resistance value. These figures evaluate experimental consistency and measurement reproducibility.

### absolute_gain_difference_vs_frequency.png

Absolute difference between repeated gain measurements as a function of frequency.

### mean_absolute_gain_difference_vs_resistance.png

Summary metric showing average repeatability error for each resistance value.
