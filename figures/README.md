# Figures

This directory contains all visualizations generated during the experimental and computational investigation of a non-ideal series RLC circuit.

The figures document the complete analysis pipeline, from raw experimental measurements to parameter reconstruction, identifiability analysis, and uncertainty quantification.

## Folder Structure

### experimental/

Figures derived directly from measured frequency-response data.

Contents include:

* Resistance sweep measurements
* Resonance and damping trends
* Source loading effects
* Experimental repeatability analysis

### reconstruction/

Figures related to parameter estimation and model fitting.

Contents include:

* Model fits to experimental data
* Residual analysis
* Effective parameter reconstruction
* Parameter drift and stability trends
* Run-to-run reconstruction comparisons

### identifiability/

Figures investigating whether model parameters can be uniquely recovered from measured frequency-response data.

Contents include:

* One-dimensional RMSE landscapes
* Two-dimensional parameter tradeoff heatmaps
* Parameter coupling analysis
* Reconstruction uncertainty bounds

## Purpose

These figures support the central research question of the project:

**How accurately can the parameters and behavior of a non-ideal dynamical system be reconstructed from experimental frequency-response measurements?**

Together, the visualizations provide evidence for model validity, parameter reconstruction performance, identifiability limitations, and uncertainty in the recovered circuit parameters.
